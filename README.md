# Basic Network Sniffer

This project is part of my Cyber Security Internship at CodeAlpha.

## 📌 Objective
To capture and analyze network packets using Python.

## 🚀 Features
- Captures live packets
- Displays source and destination IP
- Identifies protocols (TCP, UDP, ICMP)
- Packet counter
- Timestamp logging

## 🛠️ Technologies Used
- Python
- Scapy

## ▶️ How to Run

1. Install Python
2. Install Scapy:

pip install scapy

3. Run the program:

python network_sniffer.py
